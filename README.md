# nestjs-weather-app

Simple Nest.js weather app

## Getting started

```bash
$ npm install

# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
npm run start:prod
```

## Running the tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
